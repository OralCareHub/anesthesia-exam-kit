Upper molars rule

PSA + GP. On 1.6 / 2.6 add a plexus over the MB root — PSA misses it.

Upper anteriors rule

Infraorbital + NP (or GP for premolars). Infraorbital covers ASA + MSA in one shot.

Lower molars rule

Spix + Long Buccal. Spix does IAN + lingual in one puncture (T1 then withdraw 1 cm → T2).

Lower anteriors rule

Spix if you need tongue/lingual too. Mental+Incisive if only the tooth + labial. No lingual coverage from MEN.
