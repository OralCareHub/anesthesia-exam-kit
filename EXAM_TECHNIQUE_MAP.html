How to read the map

Color inside the tooth = technique for the pulp. Bands above/below = techniques for the soft tissue (palate, buccal gingiva, lingual). Alternative bands = a second technique that also works.

The two splits

Upper palate splits at the canine — front = NP, back = GP. Lower buccal splits at the mental foramen — front = MEN, back = LBN.

Spix covers everything below

Spix (IAN) anesthetizes all 16 lower teeth. Mental + Incisive is a lighter alternative for lower I / C / P only — doesn't reach molars.

Yellow MB dot = the trap

Upper M1 (1.6 / 2.6) — the mesio-buccal root escapes PSA. Always add a plexus supplement at the MB root.

⚠️ Exam Step 3 — what to say for each region

  1. Upper molar (except MB M1): "PSA / tuberosity block, plus greater palatine for the palate side."
  2. Upper M1 (1.6 / 2.6): "PSA block + plexus supplement over the MB root + greater palatine."
  3. Upper premolar / canine / incisor: "Infraorbital block (covers ASA + MSA) + nasopalatine or greater palatine for the palate side."
  4. Any single upper tooth (quick): "Plexus / supraperiosteal — bevel to bone, few mm depth, ⅓–½ cartridge."
  5. Lower molar: "Spix block (T1 for IAN → withdraw 1 cm → T2 for lingual) + long buccal for the vestibular gingiva."
  6. Lower premolar / canine / incisor: "Spix block if I need pulp + tongue; Mental+Incisive block if only the tooth + lip is needed."
  7. Palate-only work (posterior): "Greater palatine — 1 cm medial to upper M3, up+back+out, 0.2–0.3 mL, don't enter the duct."
  8. Palate-only work (anterior): "Nasopalatine — lateral to the incisive papilla, from the opposite side, 5–6 mm into the duct."
